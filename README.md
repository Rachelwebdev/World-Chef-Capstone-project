# 📖 [World Chef Conference Capstone Project] <a name="about-project"></a>

**[World Chef Conference]** is a website which showcases content and information about the upcoming World Chef summit. It gives aspiring attendants the ability to register for the event upfront, get more detailed information about the event, get in touch with planners of the event.

### Future Features <a name="future-features"></a>

**In future features, I intend to include more pages such as:**

- Program page which displays all the activities lined up on all days of the event.
- Register page which holds an attendant form for people who are interested in attending the conference.
- Sponsor page to acknowledge all sponsors of the event. This page would also give an opportunity to organizations and individuals who are willing to sponsor the event in one way or another. They can do this by filling a detailed form.
- News page containing updates about the event, speakers of the event and fun activities in the city where the conference will hold.
